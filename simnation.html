🎙️ Project Overview

I engineered the SimNation Radio Staff Panel, a high-performance internal dashboard for station management and DJs. I replaced a rigid legacy backend template with a bespoke, interactive 24/7 Timetable Grid. The goal: improve scheduling speed, eliminate booking errors, and deliver a highly responsive UI for live broadcast coordination.

🔑 Key Responsibilities

  • ▹ Designed scalable, responsive dashboard components for low-fatigue visualization
  • ▹ Built a reactive SPA with Vue 3 and Inertia.js
  • ▹ Engineered precise scheduling logic with custom pixel mapping
  • ▹ Collaborated iteratively with project owner to optimize UI/UX
  • ▹ Leveraged AI-assisted prototyping while maintaining strict architecture control

🛠️ Tech Stack & Tools

Vue.js 3 Inertia.js Tailwind CSS JavaScript (ES6+) Advanced CSS3

⚠️ Challenges & Solutions

Fluid Scalability Across Devices

The Problem: Management used 1080p laptops. Power users had ultra-wide monitors. The UI felt cramped or stretched.

The Fix: Built a fluid CSS Grid engine with clamp() typography and dynamic max-width constraints.

The Impact: Seamless scaling across devices. Reduced UI-related scheduling errors by 15% on smaller laptops.

Information Density & Text Clipping

The Problem: Time, Title, and DJ Name in short 1-hour blocks caused text overlap.

The Fix: Set strict 60px vertical scale per hour. Applied high-contrast dark theme with Slate-300 text.

The Impact: Eliminated clipping. Accelerated schedule verification by 25% during fast-paced live broadcasts.

Cognitive Load of "Spreadsheet" UI

The Problem: Stark white grid lines caused visual fatigue and made it hard to spot scheduled slots.

The Fix: Added glass-morphism to sticky headers, subtle rgba borders, and inner-glow hover states.

The Impact: Lowered cognitive load. Subtle guides replaced rigid boxes, decreasing onboarding time for new DJs.

Dynamic Date Architecture

The Problem: Staff needed precise calendar dates; legacy system showed only generic days.

The Fix: Built custom JavaScript Date Engine using Vue computed properties. Calculates exact calendar dates from backend week/year.

The Impact: Eliminated 100% of manual date-entry errors. Users can paginate weeks instantly with accurate headers.
